# Contribuer à Axent 🍪

Merci de l'intérêt que vous portez à Axent ! Ce projet est maintenu par l'Association Axent. 

**Note importante :** En soumettant une Pull Request, vous acceptez de céder les droits de vos modifications à l'Association Axent afin de maintenir l'intégrité et la propriété du projet.

## Comment contribuer ?

1. **Forkez** le dépôt sur GitHub.
2. **Clonez** votre fork localement.
3. Créez une **branche** pour votre fonctionnalité (`git checkout -b feature/ma-super-idee`).
4. **Commitez** vos changements avec des messages clairs.
5. **Pushez** sur votre branche (`git push origin feature/ma-super-idee`).
6. Ouvrez une **Pull Request**.

## Règles de conduite

- Respectez la structure de dossiers existante (Plesk compatible).
- Ne poussez jamais de clés API ou de mots de passe.
