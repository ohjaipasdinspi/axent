# 🍪 Axent — Solution de gestion du consentement

> **Ce projet est la propriété exclusive de l'Association Axent.**

Axent est une plateforme auto-hébergée permettant de gérer le consentement des cookies en conformité avec le RGPD.

Ce logiciel est publié à des fins de transparence. **Toute reproduction, copie ou utilisation commerciale est strictement interdite** sans l'accord écrit préalable de l'Association Axent. Reportez-vous au fichier `LICENSE.md` pour plus de détails.

## 🛠️ Installation
Pour installer la solution sur votre serveur Plesk, veuillez suivre le guide détaillé présent dans la documentation interne ou contacter l'association.
