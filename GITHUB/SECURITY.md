# Politique de sécurité (Security Policy)

## Versions supportées

| Version | Supportée |
| ------- | --------- |
| 1.0.x   | ✅ Oui    |
| < 1.0   | ❌ Non    |

## Signalement d'une vulnérabilité

**Ne créez pas d'issue publique** pour signaler une faille de sécurité.

Veuillez envoyer vos signalements directement par email à : **contact@axet.fr**.
Nous nous engageons à répondre sous 48h et à déployer un correctif dans les plus brefs délais.
